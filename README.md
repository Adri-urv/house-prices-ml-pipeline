# house-prices-ml-pipeline
Predicción de Precios de Vivienda – Pipeline de Machine Learning (House Prices – Kaggle)
Este proyecto implementa un pipeline clásico de Machine Learning para predecir precios de vivienda utilizando el dataset House Prices: Advanced Regression Techniques de Kaggle.
El trabajo sigue la estructura recomendada en la microcredencial de AWS: formulación del problema, análisis exploratorio, ingeniería de características, entrenamiento, evaluación y ajuste de hiperparámetros.
